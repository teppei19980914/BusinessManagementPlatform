# たすきば Knowledge Relay — Vision / 思想・価値観

本ディレクトリは、本サービス開発者の **抽象的な思い・価値観・目指す世界観** を記録する場所です。

他の `docs/` 配下が「**何を作るか / どう作るか / どう運用するか**」という具体・操作論を扱うのに対し、本ディレクトリは「**なぜ作るのか / 何を大切にするのか / どんな世界を作りたいのか**」という根源的・抽象的な問いに対する考えを残す場所として位置づけます。

機能仕様や設計判断に迷ったとき、本ドキュメントに立ち返って「自分はこのサービスで何を実現したかったのか」を再確認するための **コンパス** として機能させます。
